# dev-ruby-cli-hello-world

## dev-Description
This is a POC project to demonstrate
a classic ruby console app.

## dev-Tech stack
- perl
- ruby 3.11

## dev-Docker stack
- docker-cli
- alpine:latest
- centos:latest
- ubuntu:latest

## dev-Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## dev-To run
```sudo ./install.sh -u```

## dev-To stop (optional)
```sudo ./install.sh -d```

## dev-To see help
`sudo ./install.sh -h`
